Project: NetScope Analyzer

Description:
A basic network sniffer developed using Python, Flask, and Scapy. The application captures live network packets and displays source IP, destination IP, and protocol information through a web dashboard.

Requirements:
- Python 3.13
- Flask
- Scapy
- Npcap (Windows)

Run:
python app.py
